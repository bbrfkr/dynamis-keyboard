# dynamis
"dynamis" is the 65% keyboard integrated with trackball.

## concept
- upgradable
    - This keyboard can be updated from sandwitch mounted case to gasket mounted case.
- only one to input
    - This keyboard is integrated with trackball, so you need to input with only the one!
- compact and major
    - 65% layout that is compact and major layout have been adopted.
- rich layout options
    - split backspace, split left and right shift, split space and iso enter are supported.
